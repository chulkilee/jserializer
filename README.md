# Jserializer

jserializer is simple serializer using [jbuilder](https://rubygems.org/gems/jbuilder)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jserializer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jserializer
